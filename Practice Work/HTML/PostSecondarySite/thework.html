The Workplace

What is the Workplace like for a Computer Programmer?

Computer programmers usually work in offices, most commonly in the computer systems design and related services industry. Programmers normally work alone, but sometimes work with other computer specialists on large projects. Because writing code can be done anywhere, many programmers telecommute.

Many companies offer benefits to their computer programming employees. Some of these benefits may be:

  • Dental Insurance
  • Extended Health Care
  • Ability to Work from Home
  • Wellness Programs
  • Company Events
  • Etc...

Related Jobs

Computer programming has a lot of transferable skills. Thus, computer programmers can branch off into multiple careers.

  • Applications Developer
  • Computer Programmer Analyst
  • Software Developer
  • Junior Game Developer
  • Junior Web Developer
  • Junior Database Designer
  • Software Tester
  • Technical Support Representative

Expected Salary

Salaries for computer programmers are fairly decent for both entry and the median level salaries. Computer programmers with many years of work experience will find themselves at a higher salary ranging up to $100k+.
